This repository contains the backend code for my project Quiz App. Quiz App is an website app that allows users to create, share, and play quizzes. It includes features that allow users to upload images for their quizzes (eg. cover images, question images) and view their past quiz attempts. User's can also share information about themselves through a profile page. The backend was built using django and the django rest framework (DRF), and the database uses MySQL The DRF gave me access to serializers for input validation, JWT tokens for autentication and authorization, the ability to build custom permission classes, and request body parsing.
